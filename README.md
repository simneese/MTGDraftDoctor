# MTG Draft Doctor
MTG limited formats web based deck building tool - can be used for building draft or sealed decks. Analyze your current mana curve and get suggestions for cards to cut or add based on mana value! Then, get recommended basic lands to add to your deck based on cost pip counts.

## How-To
### Basic
1. [optional] adjust the settings on the mana curve tool to make adjustments to the ideal curve ranges.
2. Input number of cards in your deck for each mana cost bucket.
3. Recommended adds or cuts appear to the side of your mana curve.
4. Count the number of nonbasic lands in your deck and input that number into the mana base tool.
5. Count the total number of each color pip which appears in the cost of the cards. Input them into the mana base tool.
6. Count the amount of mana provided by your nonbasic lands and input that into the mana base tool.
7. Recommended basic land counts for your deck appear to the side of the mana base tool.

### Advanced
1. [optional] use set filters to limit the cards which appear in the search (based on which sets you are playing)
2. Search for the cards in your deck. Add them to the deck and set the number which appear in your deck.
3. Set the number of additional basic lands you will need to add to your deck.
4. [optional] expand the deck preview to see how your cards are grouped based on mana cost.
5. Mana curve and mana base tools are automatically populated based on cards in your deck.
6. Use recommendations to help decide what to cut or add to your deck.
